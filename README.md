# Data Visualization Projects

This repository contains data visualization projects organized into project-based subfolders.

## Projects

### 📱 [Screen_Time_Analysis](./Screen_Time_Analysis/)
Analysis of screen time data across different apps.
- `ScreenTimeAnalysis.ipynb` — Jupyter Notebook with the analysis
- `Screentime-App-Details.csv` — Dataset with app-level screen time details

### ⌚ [Smartwatch_Data_Analysis](./Smartwatch_Data_Analysis/)
Analysis of daily activity data collected from smartwatch devices.
- `Smartwatch Data Analysis.ipynb` — Jupyter Notebook with the analysis
- `dailyActivity_merged.csv` — Dataset with merged daily activity records

### 📊 [Unemployment_Analysis](./Unemployment_Analysis/)
Analysis and visualization of unemployment trends.
- `UnemploymentAnalysis.ipynb` — Jupyter Notebook with the analysis

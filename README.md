# Data Visualization Projects

This repository contains data visualization projects organized into project-based subfolders.

## Projects

### 📱 Screen_Time_Analysis
Analysis of screen time data across different apps.
- `ScreenTimeAnalysis.ipynb` — Jupyter notebook with analysis and visualizations
- `Screentime-App-Details.csv` — Dataset with app-level screen time details

### ⌚ Smartwatch_Data_Analysis
Analysis of smartwatch activity data.
- `Smartwatch Data Analysis.ipynb` — Jupyter notebook with analysis and visualizations
- `dailyActivity_merged.csv` — Dataset with daily activity records

### 📉 Unemployment_Analysis
Analysis and visualization of unemployment trends.
- `UnemploymentAnalysis.ipynb` — Jupyter notebook with analysis and visualizations
